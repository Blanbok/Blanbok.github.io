JavaScript Study Note

Aug 9, 2017 16:21 · 139 words · 1 minute read JavaScript-Study

刚开始学习，也不敢太贪进，便从初级的教材《JavaScript从入门到精通（第2版）》开始学起。

首先接触的，自然是熟悉的“Hello World”了。

    javascript:alert("Hello World");

其中

javascript作为标签

alert()作为命令 执行弹出框操作

"Hello World"作为命令的输入项

以此三者，形成完整的弹出框操作。
