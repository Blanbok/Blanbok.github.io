JavaScript Study Note

Aug 9, 2017 16:21 · 939 words · 2 minutes read JavaScript-Study

刚开始学习，也不敢太贪进，便从初级的教材《JavaScript从入门到精通（第2版）》开始学起。

首先接触的，自然是熟悉的“Hello World”了。

    javascript:alert("Hello World");

其中

javascript作为标签

alert()作为命令 执行弹出框操作

"Hello World"作为命令的输入项

以此三者，形成完整的弹出框操作。

在HTML中使用javaScript

	<script type="text/javascript">
		document.write("hello");
		document.write(" world");
	</script>

标签：script

标签属性：type="text/javascript"

然后执行文档写入操作

对象：document

函数：write

参数："hello"

基本数据类型

  • 数字（Number）
  • 字符串（String）
  • 布尔值（Boolean）
  • Null
  • 未定义（Undefined）
  • 对象（Object）

数值函数 isNaN()

“is not a number”

是数字返回false 非数字返回true

Math对象

返回的值类型：

  • Math.PI pi值
  • Math.pow(x,y)返回x的y次幂
  • Math.random()返回一个伪随机数
  • Math.abs(x) 返回x的绝对值
  • Math.round(x)将x取值为最接近的整数值

转义字符序列

转义符号 序列值
\b 退格符
\t 水平制表符
\n 换行符
\v 垂直制表符
\f 换页符
\r 回车符
\ 反斜杠符

字符串

var str = "this is a string";
alert(str.length);

直接用点语法获取字符串的长度

使用str.substring(3,9)表示从第3位字符开始到达第9位字符，s是第0位字符

使用str.slice(3,9)表示从第3位字符,向后数9位字符

使用str.toUppearCase()将字符串中所有的字符转为大写

使用str.toLowerCase()将字符串中所有的字符转为小写

数组

直接创建数组的方法

var star = ["polaris","Deneb","Vega","Altair"];

var star = [["Polaris","Ursa Minor"],["Deneb","Cygnus"],["Vega","Lyra"],["Altair","Aquila"]];

Date

get方法

方法名称 说明
getDate() 返回月中的某天
getDay() 返回星期中的某天
getFullYear() 返回四位数年份表示，相较于getYear()方法在大多数环境下被推荐使用
getHours() 返回日期中的小时
getMilliseconds() 返回日期中的毫秒
getMinutes() 返回日期中的分钟
getMonth() 返回日期中的月
getSeconds 返回日期中的秒
getTime() 返回自1970年1月1日起的毫秒
getTimeZoneOffset() 返回UTC和本地时间之差值所对应的分钟数

set方法

方法名称 说明
setDate() 设置月中的某天
setFullYear() 设置日期中的四位数年份表示。也接受月以及月中的一天所对应的整数
setHours() 设置日期中的小时
setMilliseconds() 设置日期中的毫秒
setMinutes() 设置日期中的分钟
setMonth() 设置日期中的月
setSeconds() 设置日期中的秒
setTime() 使用自1970年1月1日起的毫秒数设置时间
